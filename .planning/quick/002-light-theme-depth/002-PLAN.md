# Quick Task 002: Add Visual Depth to Light Theme

## Objective
Add visual depth to the light theme through gradients, enhanced shadows, and glass/blur effects to make it feel more modern and premium.

## Tasks

### Task 1: Enhanced Shadow System
- Update light theme shadows to be more pronounced
- Add colored shadows with violet tint for depth
- Create layered shadow effects for cards and modals

### Task 2: Glass/Blur Effects for Light Theme
- Add backdrop-filter blur to cards and popovers
- Create frosted glass effect for modals
- Add subtle transparency to surfaces

### Task 3: Gradient Accents
- Add subtle gradients to primary buttons
- Create gradient borders for focused elements
- Add ambient glow effects on hover
